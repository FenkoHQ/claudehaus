Setup Guide
1. Add HTTP hooks to ~/.claude/settings.json: 
{
  "hooks": {
    "PreToolUse": [{ "matcher": "*", "hooks": [
      { "type": "http",
        "url": "http://127.0.0.1:8420/api/hooks/PreToolUse",
        "headers": {"Authorization": "Bearer $CLAUDEHAUS_TOKEN"},
        "allowedEnvVars": ["CLAUDEHAUS_TOKEN"] }
    ]}],
    "PostToolUse": [{ "matcher": "*", "hooks": [
      { "type": "http",
        "url": "http://127.0.0.1:8420/api/hooks/PostToolUse",
        "headers": {"Authorization": "Bearer $CLAUDEHAUS_TOKEN"},
        "allowedEnvVars": ["CLAUDEHAUS_TOKEN"] }
    ]}],
    "PermissionRequest": [{ "matcher": "*", "hooks": [
      { "type": "http",
        "url": "http://127.0.0.1:8420/api/hooks/PermissionRequest",
        "headers": {"Authorization": "Bearer $CLAUDEHAUS_TOKEN"},
        "allowedEnvVars": ["CLAUDEHAUS_TOKEN"],
        "timeout": 300 }
    ]}],
    "Stop": [{ "hooks": [
      { "type": "http",
        "url": "http://127.0.0.1:8420/api/hooks/Stop",
        "headers": {"Authorization": "Bearer $CLAUDEHAUS_TOKEN"},
        "allowedEnvVars": ["CLAUDEHAUS_TOKEN"] }
    ]}]
  }
}
2. Set your token in the shell: 
export CLAUDEHAUS_TOKEN="your-token-here"

No script required. Claude Code posts directly to the server.

1. Set environment variables: 
export CLAUDEHAUS_TOKEN="your-token-here"
export CLAUDEHAUS_URL="http://127.0.0.1:8420"
2. Copy the hook script to your PATH: 
cp scripts/claudehaus-hook ~/.local/bin/
3. Add hooks to ~/.claude/settings.json: 
{
  "hooks": {
    "PermissionRequest": [{ "matcher": "*", "hooks": [
      { "type": "command", "command": "claudehaus-hook" }
    ]}],
    "PreToolUse": [{ "matcher": "*", "hooks": [
      { "type": "command", "command": "claudehaus-hook" }
    ]}],
    "Notification": [{ "hooks": [
      { "type": "command", "command": "claudehaus-hook" }
    ]}],
    "SessionStart": [{ "hooks": [
      { "type": "command", "command": "claudehaus-hook" }
    ]}]
  }
}
4. To chain with existing hooks: 
"command": "claudehaus-hook --chain '/path/to/your-hook.sh'"

Command hooks support all events including Notification and SessionStart.
